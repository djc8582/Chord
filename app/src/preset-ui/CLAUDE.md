# preset-ui

> **Tier 3** — Depends on `document-model`, `browser`.

## What This Is

Preset browser, save/load, snapshot system, morphing UI.

## Key Features
- Per-node preset browser (load/save/init/randomize/A-B compare)
- Patch-level snapshot system (capture/recall/morph)
- Preset import from other formats (.fxp, .vstpreset)
- Community preset browsing

## Dependencies
- `document-model` (Tier 0), `browser` (Tier 2)
