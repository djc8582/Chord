# audio-editor

> **Tier 4** — Depends on `timeline`, `document-model`.

## What This Is

Destructive waveform editor. Cut, splice, normalize, reverse, fade, spectral editing. Opens when double-clicking audio clips.

## Dependencies
- `timeline` (Tier 3), `document-model` (Tier 0)

## Definition of Done
- [ ] Opens from double-clicking audio clip
- [ ] Selection and basic edit operations (cut, copy, paste, delete)
- [ ] Normalize, reverse, fade in/out
- [ ] Undo/redo history
- [ ] Spectral view (display only for MVP, editing in Phase 2)
