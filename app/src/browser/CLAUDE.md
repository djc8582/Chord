# browser

> **Tier 2** — Depends on `document-model`.

## What This Is

Node library browser, file browser, and community patch browser. The left panel.

## Key Features
- Node library: categorized list of all available node types with search
- File browser: project files, audio files, sample libraries
- Community browser: search and preview community patches (requires cloud)
- Drag-and-drop: drag nodes from library to canvas, drag files to nodes

## Dependencies
- `document-model` (Tier 0) — node type definitions
