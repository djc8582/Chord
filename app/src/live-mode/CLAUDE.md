# live-mode

> **Tier 4** — Depends on `canvas`, `preset-ui`, `controller-integration`.

## What This Is

Fullscreen performance mode. Custom layouts, big controls, setlist, panic button, performance lock.

## Dependencies
- `canvas` (Tier 1), `preset-ui` (Tier 3), `controller-integration` (Tier 3)

## Definition of Done
- [ ] Fullscreen mode with no editing UI
- [ ] Customizable widget layout (knobs, sliders, meters)
- [ ] Setlist with next/prev navigation
- [ ] Panic button (all mute + MIDI all-notes-off)
- [ ] Performance lock (ignore mouse, only mapped controllers work)
