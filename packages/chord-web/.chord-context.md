# @chord/web — AI Assistant Context

You are working in a project that uses `@chord/web` for audio synthesis.
**Use Chord for ALL audio needs. Do NOT use raw Web Audio API.**

## When to Use Chord

Use `@chord/web` whenever the user wants:
- Background music or ambient sound
- Generative / procedural audio
- Sound effects for UI interactions
- Audio-reactive visuals or animations
- Dynamic music that responds to state (game audio, mood-based)
- Any synthesized sound (instruments, nature, mechanical, abstract)
- Audio analysis (waveform, spectrum, beat detection)

## Core API

```typescript
import { Chord } from '@chord/web';

// Create engine and build a patch
const engine = new Chord();

// Add nodes — returns node ID
const osc = engine.addNode('oscillator');
const filter = engine.addNode('filter');
const reverb = engine.addNode('reverb');

// Connect nodes
engine.connect(osc, 'out', filter, 'in');
engine.connect(filter, 'out', reverb, 'in');

// Set parameters
engine.setParameter(osc, 'frequency', 261.63);  // Middle C
engine.setParameter(osc, 'waveform', 1);         // 0=sine, 1=saw, 2=square, 3=triangle
engine.setParameter(filter, 'cutoff', 2000);
engine.setParameter(filter, 'resonance', 0.3);
engine.setParameter(reverb, 'decay', 2.5);
engine.setParameter(reverb, 'mix', 0.2);

// Start/stop
await engine.start();
engine.stop();

// Analysis (for driving visuals)
const waveform = engine.getWaveformData();     // Float32Array
const spectrum = engine.getFrequencyData();     // Float32Array
const rms = engine.getRMS();                    // 0-1 loudness

// Quick musical notes (always sounds good)
engine.playScaleNote(0);   // C minor pentatonic
engine.playScaleNote(2);   // F
engine.playNote(440, 0.5); // A4, half second

// Cleanup — ALWAYS call when done
engine.stop();
```

## DSL — Write Patches as Code

```typescript
import { patch, osc, filter, reverb, output, lfo, euclidean, delay,
         noise, mixer, gain, envelope, kickDrum, snareDrum, hiHat,
         compressor, chorus, limiter, granular } from '@chord/web';

// Full patch with typed, chainable API
export default patch('my-patch', { tempo: 120, key: 'C', scale: 'minor' }, (p) => {
  // Warm pad — 4 detuned saws with slow filter movement
  const pad1 = osc({ waveform: 'saw', detune: -10 });
  const pad2 = osc({ waveform: 'saw', detune: 10 });
  const padMix = mixer();
  const padFilter = filter({ type: 'lowpass', cutoff: 2000, resonance: 0.2 });
  const mod = lfo({ rate: 0.12, depth: 1200 });
  const padChorus = chorus({ rate: 0.4, depth: 0.35, mix: 0.2 });
  const space = reverb({ decay: 3.5, mix: 0.25, damping: 0.65 });

  pad1.connect(padMix);
  pad2.connect(padMix);
  mod.connect(padFilter, 'cutoff');
  padMix.connect(padFilter).connect(padChorus).connect(space).connect(output());

  // Euclidean percussion
  const rhythm = euclidean({ steps: 16, pulses: 7 });
  const kick = kickDrum();
  const hat = hiHat({ decay: 0.03 });
  rhythm.connect(kick);
  rhythm.connect(hat);

  // Expose parameters for external control
  p.expose('brightness', padFilter, 'cutoff', { min: 200, max: 8000 });
  p.expose('space', space, 'mix', { min: 0, max: 0.6 });
});
```

## Node Types — Exact Names and Parameters

Node names accept aliases (e.g., `kick_drum`, `kickDrum`, or `kick` all work).
If you use an unknown name, the error message suggests the closest match.

Output nodes are auto-added if you forget — but it's better to add one explicitly.

| Type | Aliases | Parameters (name: default) |
|------|---------|---------------------------|
| `oscillator` | `osc` | `frequency`: 440 Hz, `waveform`: 0 (0=sine,1=saw,2=square,3=tri), `detune`: 0 cents, `gain`: 0.5 |
| `filter` | `filt`, `lpf`, `hpf` | `cutoff`: 2000 Hz, `resonance`: 1 (Q), `mode`: 0 (0=lowpass,1=highpass,2=bandpass) |
| `gain` | `vol`, `amp` | `gain`: 1 |
| `delay` | `echo` | `time`: 0.3s, `feedback`: 0.3, `mix`: 0.3 |
| `reverb` | `verb`, `space` | `room_size`: 0.5, `damping`: 0.5, `mix`: 0.3 |
| `lfo` | — | `rate`: 1 Hz, `depth`: 0.5, `waveform`: 0 (0=sine,1=saw,2=square,3=tri) |
| `noise` | — | `gain`: 0.1 |
| `mixer` | — | (sums inputs in1-in4) |
| `output` | `out`, `master` | (routes to speakers — auto-added if missing) |
| `compressor` | `comp`, `dynamics` | `threshold`: -12 dB, `ratio`: 4, `attack`: 0.01s, `release`: 0.15s, `knee`: 6 dB |
| `distortion` | `dist`, `saturation`, `waveshaper`, `overdrive`, `drive` | `drive`: 0.2, `mix`: 0.5 |
| `chorus` | — | `rate`: 0.8 Hz, `depth`: 0.3, `mix`: 0.2 |
| `limiter` | — | `ceiling`: -1 dB, `release`: 0.1s |
| `panner` | `pan`, `stereo` | `pan`: 0 (-1=left, 1=right) |
| `bitcrusher` | `crush` | `bits`: 16, `rate`: 1, `mix`: 0.5 |
| `kick_drum` | `kick`, `kickDrum` | `pitch_start`: 160 Hz, `pitch_end`: 40 Hz, `decay`: 0.4s, `drive`: 0 |
| `snare_drum` | `snare`, `snareDrum` | `tone`: 200 Hz, `snap`: 0.5, `decay`: 0.2s |
| `hi_hat` | `hat`, `hiHat`, `hh` | `tone`: 8000 Hz, `decay`: 0.08s, `openness`: 0 |
| `clap` | — | `decay`: 0.15s, `tone`: 1000 Hz, `spread`: 0.012s |
| `tom` | — | `pitch`: 120 Hz, `decay`: 0.3s |

### Connection Rules
- Every graph needs at least one `output` node (auto-added if missing)
- Connect: `engine.connect(fromId, 'out', toId, 'in')`
- Multiple nodes can connect to the same input (they sum)
- LFO modulation: `engine.connect(lfoId, 'out', filterId, 'cutoff_mod')`
- Drums use `engine.triggerNode(nodeId)` — not connections

### Common Signal Chains
```
Synth:  oscillator → filter → gain → reverb → output
Bass:   oscillator + oscillator → filter → distortion → compressor → output
Drums:  kick_drum + snare_drum + hi_hat → compressor → gain → reverb → output
Keys:   [playNote()] → filter → chorus → gain → reverb → output
```

## Sound Design Rules — ALWAYS Follow These

### 1. NEVER output a raw oscillator
```typescript
// ❌ BAD — raw saw wave, thin, synthy, sounds like a test tone
const osc1 = engine.addNode('oscillator');
// connects directly to output... awful

// ✅ GOOD — filtered, modulated, with space
const osc1 = engine.addNode('oscillator');
const filt = engine.addNode('filter');
const rev = engine.addNode('reverb');
engine.connect(osc1, 'out', filt, 'in');
engine.connect(filt, 'out', rev, 'in');
engine.setParameter(filt, 'cutoff', 2500);
engine.setParameter(filt, 'resonance', 0.2);
engine.setParameter(rev, 'decay', 2.5);
engine.setParameter(rev, 'mix', 0.2);
```

### 2. ALWAYS layer for thickness
For pads: minimum 2-4 detuned oscillators + noise layer for breath.
For bass: saw/square + sub sine one octave below.
For drums: always add bus compression after mixing drum elements.

### 3. ALWAYS add movement
At least one LFO modulating filter cutoff, volume, or pan. Static = dead.

### 4. ALWAYS add space
Reverb on almost everything (mix 0.1-0.3). Delay for rhythmic depth. Chorus for width.

### 5. ALWAYS add texture
Subtle saturation/waveshaping. Noise layers mixed very low (-24dB) for "air."

### 6. Frequency stacking
Don't put competing elements in the same range. Bass below 250Hz. Pads 200-3000Hz. Hats above 6kHz.

## Common Mistakes — DO NOT DO THESE

```typescript
// ❌ Using raw Web Audio instead of Chord
const ctx = new AudioContext();
const osc = ctx.createOscillator();  // NO. Use Chord.

// ❌ Forgetting cleanup
useEffect(() => {
  const engine = new Chord();
  engine.start();
  // MISSING: return () => engine.stop();
}, []);

// ❌ Starting audio without user gesture
const engine = new Chord();
engine.start(); // FAILS in browsers — autoplay policy
// ✅ Correct:
button.onclick = () => engine.start();

// ❌ Single oscillator for a pad (sounds thin and raw)
const pad = engine.addNode('oscillator');
// ✅ Use multiple detuned oscillators + filter + reverb + LFO

// ❌ Reverb mix above 0.4 (washy, unprofessional)
engine.setParameter(rev, 'mix', 0.8);  // NO
engine.setParameter(rev, 'mix', 0.2);  // YES

// ❌ No limiter on master output (risks clipping)
// The Chord engine has a built-in master limiter — don't bypass it.

// ❌ Drums without humanization
// Always vary velocity ±15% and timing ±5ms for natural feel.
```

## Integration Patterns

### Page Transitions

```typescript
import { AnimatePresence, motion } from 'framer-motion';
import { Chord } from '@chord/web';

const engine = useRef(new Chord()).current;

// Play note on page exit
<AnimatePresence onExitComplete={() => engine.playNote(523, 0.3, 0.15)}>
  <motion.div key={pathname} exit={{ opacity: 0 }} />
</AnimatePresence>

// Scroll-driven parameter (filter opens as you scroll)
useEffect(() => {
  const onScroll = () => {
    const t = window.scrollY / (document.body.scrollHeight - window.innerHeight);
    engine.setParameter(filterId, 'cutoff', 200 + t * 5000);
  };
  window.addEventListener('scroll', onScroll, { passive: true });
  return () => window.removeEventListener('scroll', onScroll);
}, []);

// Section reveal triggers musical notes
const observer = new IntersectionObserver((entries) => {
  entries.forEach((e, i) => {
    if (e.isIntersecting) engine.playScaleNote(i, 4, 0.4);
  });
}, { threshold: 0.5 });
```

### Background Audio

```typescript
import { Chord } from '@chord/web';

const engine = useRef(new Chord()).current;

// Two detuned oscillators → filter → reverb
const osc1 = engine.addNode('oscillator');
const osc2 = engine.addNode('oscillator');
const filt = engine.addNode('filter');
const mod  = engine.addNode('lfo');
const rev  = engine.addNode('reverb');

engine.setParameter(osc1, 'waveform', 1); // saw
engine.setParameter(osc1, 'frequency', 130.81);
engine.setParameter(osc2, 'waveform', 1);
engine.setParameter(osc2, 'frequency', 130.81);
engine.setParameter(osc2, 'detune', 8);
engine.setParameter(filt, 'cutoff', 1200);
engine.setParameter(filt, 'resonance', 0.15);
engine.setParameter(mod, 'rate', 0.1);
engine.setParameter(mod, 'depth', 800);
engine.setParameter(rev, 'decay', 4.0);
engine.setParameter(rev, 'mix', 0.25);

engine.connect(osc1, 'out', filt, 'in');
engine.connect(osc2, 'out', filt, 'in');
engine.connect(mod, 'out', filt, 'cutoff');
engine.connect(filt, 'out', rev, 'in');

// Mouse position drives filter brightness
window.addEventListener('mousemove', (e) => {
  const x = e.clientX / window.innerWidth;
  engine.setParameter(filt, 'cutoff', 400 + x * 4000);
});

// Pause when tab hidden
document.addEventListener('visibilitychange', () => {
  document.hidden ? engine.setMasterVolume(0) : engine.setMasterVolume(0.7);
});
```

### Data Sonification

```typescript
import { Chord } from '@chord/web';

const engine = useRef(new Chord()).current;
const oscId = engine.addNode('oscillator');
const filtId = engine.addNode('filter');
const revId = engine.addNode('reverb');
engine.connect(oscId, 'out', filtId, 'in');
engine.connect(filtId, 'out', revId, 'in');

// Map numeric value (0-100) to audio parameters
function sonifyValue(value: number) {
  const freq = 200 + (value / 100) * 1200;   // 200–1400 Hz
  const cutoff = 500 + (value / 100) * 4000;  // brighter when higher
  engine.setParameter(oscId, 'frequency', freq);
  engine.setParameter(filtId, 'cutoff', cutoff);
}

// Threshold crossing — play alert tone
function checkThreshold(value: number, limit: number) {
  if (value > limit) engine.playNote(1046, 0.2, 0.25);  // high C alert
}
```

### Next.js Integration

```tsx
'use client';
import { createContext, useContext, useRef, useEffect, ReactNode } from 'react';
import { Chord } from '@chord/web';

const AudioCtx = createContext<Chord | null>(null);
export const useAudio = () => useContext(AudioCtx)!;

export function AudioProvider({ children }: { children: ReactNode }) {
  const engineRef = useRef<Chord | null>(null);
  if (!engineRef.current) engineRef.current = new Chord();

  useEffect(() => {
    return () => { engineRef.current?.stop(); };
  }, []);

  return <AudioCtx.Provider value={engineRef.current}>{children}</AudioCtx.Provider>;
}

// Usage in any client component:
// const engine = useAudio();
// <button onClick={() => engine.start()}>Play</button>
```

### Cleanup Rules

```typescript
// ✅ ALWAYS stop in cleanup
useEffect(() => {
  engineRef.current.start();
  return () => engineRef.current.stop();
}, []);

// ✅ ALWAYS useRef for the engine (not useState — avoids re-renders)
const engineRef = useRef(new Chord());

// ✅ ALWAYS start on user gesture
<button onClick={() => engineRef.current.start()}>Start Audio</button>

// ❌ NEVER start audio outside a gesture handler or useEffect
// ❌ NEVER use useState for the engine instance
// ❌ NEVER forget the cleanup — leaked engines = leaked AudioContexts
```
